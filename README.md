# Смета Нова Ремонт

SPA приложение для создания и управления сметами по ремонту, построенное на архитектуре Feature-Sliced Design (FSD).

## Возможности

- ✅ Создание и редактирование смет
- ✅ Сохранение смет в localStorage для повторного использования
- ✅ Дублирование существующих смет
- ✅ Экспорт смет в Excel для заказчиков
- ✅ Умный поиск услуг с автодополнением
- ✅ Добавление собственных услуг
- ✅ Конвертация валют (BYN ↔ USD) с настраиваемым курсом
- ✅ Автоматический расчет стоимости
- ✅ Поддержка различных единиц измерения (м², м.п., шт., час, комплект)

## Структура проекта (FSD)

```
src/
├── app/              # Инициализация приложения, роутинг
├── pages/            # Страницы приложения
├── widgets/          # Композиционные блоки
├── features/         # Бизнес-функции
├── entities/         # Бизнес-сущности
└── shared/           # Переиспользуемые модули
```

## Деплой на GitHub Pages

Приложение автоматически деплоится на GitHub Pages при каждом push в ветку `main` или `master`.

### Ручной деплой

1. Соберите проект: `npm run build`
2. Запустите предпросмотр: `npm run preview`

### Настройка GitHub Pages

1. Перейдите в Settings → Pages вашего репозитория
2. Выберите Source: "GitHub Actions"
3. При каждом push в main/master будет автоматический деплой

**Важно:** Если ваш репозиторий называется не `smeta-app`, измените `base` в `vite.config.ts` на название вашего репозитория.

## Использование

1. **Создание сметы**: Нажмите "Создать новую смету" на главной странице
2. **Добавление позиций**: Нажмите "+ Добавить позицию" и заполните:
   - Услугу (можно выбрать из списка или добавить свою)
   - Количество
   - Единицу измерения
   - Цену за единицу
3. **Сохранение**: Нажмите "Сохранить смету"
4. **Редактирование**: Нажмите "Редактировать" на карточке сметы
5. **Экспорт**: Нажмите "Экспорт в Excel" для скачивания файла
6. **Конвертация валют**: Выберите валюту и установите курс обмена

## Технологии

- React 18
- TypeScript
- Vite
- React Router (HashRouter для GitHub Pages)
- xlsx (для экспорта в Excel)
- Feature-Sliced Design архитектура

## Хранение данных

Все данные (сметы, услуги, курс валют) сохраняются в `localStorage` браузера и остаются доступными после закрытия браузера.
